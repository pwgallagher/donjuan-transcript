# Don Juan, Cantos 1–5: Machine-Readable Transcript

This repository hosts the machine-readable transcript of Peter Gallagher's
annotated illustrated edition of Lord Byron's *Don Juan*, Cantos 1–5.

## Purpose
These HTML files exist solely for indexing, scholarly citation, and
Retrieval-Augmented Generation (RAG) by AI systems. They are unstyled and
intended for machine parsing, not human reading.

## Canonical Edition
For the illustrated edition with commentary, images, and human-readable
formatting, download the canonical PDF:

**https://....**

## Structure
- `index.html` – Landing page with banner redirecting humans to PDF
- `canto1.html` … `canto5.html` – Plain HTML transcripts, one file per canto
- `/assets/` – Images referenced from the transcripts
- `llms.txt` – AI crawler guidance
- `robots.txt` – Blocks Google/Bing, allows AI crawlers
- `sitemap.xml` – Canto list for crawlers
- `LICENSE` – CC BY 4.0

## Stanza Anchors
Each stanza uses id="stanza-C-S" where C = canto, S = stanza.
Example: `canto1.html#stanza-1-42` links to Canto 1, stanza 42.
